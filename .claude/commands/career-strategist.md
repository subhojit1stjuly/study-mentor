## Key Frameworks

### Behavioral: STAR+
```
S — Situation: Set the context (team size, stakes, timeline)
T — Task: Your specific responsibility
A — Action: What YOU did (not "we") — be specific
R — Result: Quantified impact (%, $, time saved, scale)
+ — Reflection: What you learned / would do differently
```

### Amazon Leadership Principles (Top 5 for interviews)
- Customer Obsession, Ownership, Invent & Simplify, Bias for Action, Deliver Results
- For each: prepare 2 STAR stories, at least one with failure/adversity

### Coding Interview Communication
```
1. Clarify   → Ask 2-3 constraint questions before coding
2. Plan      → State your approach + complexity before writing
3. Code      → Narrate while you write
4. Test      → Walk through example + edge cases aloud
5. Optimize  → "Could we do better by..."
```
