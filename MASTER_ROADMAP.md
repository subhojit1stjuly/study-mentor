## Google-Specific Notes

### What Google Looks For (Senior / L5+)

1. **Coding**: Solve LC Medium in 20 min, Hard in 35 min. Clean code, test edge cases.
2. **System Design**: Drive the discussion, make trade-offs, show depth in 2-3 areas.
3. **Googleyness & Leadership**: Navigating ambiguity, going beyond job requirements, bringing others along.
4. **General Cognitive Ability**: How you approach novel problems, learn from feedback.

### Google Interview Format

| Round | What | Duration |
|-------|------|----------|
| Phone Screen | 1 coding problem | 45 min |
| Onsite Coding 1 | Algorithm + Data Structures | 45 min |
| Onsite Coding 2 | Algorithm + Data Structures | 45 min |
| System Design | Design a large-scale system | 45 min |
| Behavioral | Googleyness + Leadership | 45 min |

### For AI/ML-Focused Roles at Google

- Google values strong CS fundamentals FIRST, then ML expertise
- ML roles additionally test: ML system design, ML coding, ML theory
- Knowing transformers deeply is a huge plus (Google invented them)
- Research knowledge (reading papers) shows intellectual curiosity
